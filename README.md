# 📰 Newsy

> Delivering the freshest news, straight from the source. 

Newsy is a dynamic, fast, and responsive news website that fetches real-time articles from the NewsAPI. Stay updated with the latest headlines, filter by categories, and dive deep into the stories that matter most.

---

## ✨ Features

* **Real-Time Data:** Fetches the latest news articles instantly via API.
* **Responsive Design:** Looks clean and works flawlessly on desktop, tablet, and mobile.
* **Sleek UI:** Built with an intuitive and modern user interface.

---

## 🛠️ Tech Stack

* **Frontend:** HTML & Vanilla JS
* **Styling:** Tailwind CSS / Styled Components / CSS3 / Bootstrap
* **API:** NewsAPI
* **Deployment:** Render
